# Exploratory-Data-Analysis
Short EDA project
Exploratory data analysis involves summarizing datasets to uncover patterns, relations and insights.
Steps involved are as following - 
1. Data Cleaning:
   a) Handle missing values
   b) Remove duplicates
   c) Fix inconsistencies
2. Data Transformation:
   a) Changing data type
   b) Normalize data
   c) Creating new features if needed
3. Aggregation and Filtering:
   a) Summarizing data
   b) Applying filters to focus on specific aspects
